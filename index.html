CSS-Reloader by auchenberg

CSS Reloader is an firefox extension that allows you to reload all the CSS of any site without you have to reload the page itself.

Contact

(kenneth@auchenberg.dk)

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/auchenberg/CSS-Reloader

get the source code on GitHub : auchenberg/CSS-Reloader
